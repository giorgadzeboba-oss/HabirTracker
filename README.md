# 🎯 Habit Tracker - ჩვევების თრექერი

ეს არის ანდროიდ აპლიკაცია, რომელიც მომხმარებლებს ეხმარება ყოველდღიური ჩვევების ჩამოყალიბებასა და მათ კონტროლში.

## 🚀 ფუნქციონალი
- **ჩვევების მართვა**: დამატება, რედაქტირება და წაშლა.
- **პროგრესის თრექინგი**: ყოველდღიური სტატისტიკა და შესრულების ისტორია.
- **კატეგორიები**: ჩვევების დაჯგუფება კატეგორიების მიხედვით (ჯანმრთელობა, პროდუქტიულობა და ა.შ.).
- **ძიება და ფილტრაცია**: ჩვევების მარტივი ძიება და კატეგორიებით გაფილტვრა.
- **შეტყობინებები (Reminders)**: WorkManager-ის გამოყენებით მომხმარებელს მისდის შეხსენება ჩვევის შესრულებაზე.

## 🛠 ტექნიკური დეტალები
- **Jetpack Compose**: თანამედროვე UI დეკლარატიული მიდგომით.
- **MVVM არქიტექტურა**: სუფთა და ტესტირებადი კოდის სტრუქტურა.
- **Room Database**: მონაცემების ლოკალურად შესანახად.
- **Dagger Hilt**: Dependency Injection-ისთვის.
- **WorkManager**: ფონური პროცესების სამართავად (შეტყობინებების დაგეგმვა).
- **Navigation Compose**: ეკრანებს შორის ნავიგაციისთვის.
